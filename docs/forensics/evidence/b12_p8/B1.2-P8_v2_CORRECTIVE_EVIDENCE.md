## Remediation Implemented
### A) Constant-work webhook auth at dependency choke point
- Added provider-specific auth dependencies that:
  1. read bounded raw body
  2. resolve tenant secret if present
  3. use dummy secret when not found
  4. always execute provider verify function
  5. authorize only when `(secret_found && signature_valid)`
  6. never pass dummy tenant context downstream
